# ZimuDog_Server
用python爬取字幕库的的字幕信息

## 如何使用
在ubuntu的shell里面:
1. 请定位到 ZimuKu.py 这个文件的位置
2. 请输入``python ZimuKu.py``
3. 在交互式的命令行中输入你要搜索的电影名称
